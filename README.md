## O que o projeto faz: 
O sistema tem como função ser um assistente de IA para a monitoração de uma missão espacial, podendo receber dados da missão como a temperatura, status do sinal energia e status dos módulos de operação, com as informações recolhidas gera um pequeno relatório de status atual e realiza uma ação automática caso necessária a intervenção.
